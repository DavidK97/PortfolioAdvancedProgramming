Hvad sker der nu?
Din Java-kode
int n = arr.length;
for (int i = 0; i < n - 1; i++) {
    for (int j = 0; j < n - i - 1; j++) {
        if (arr[j] > arr[j + 1]) {
            swap(arr, j, j + 1);
        }
    }
}
Pointer og Farveforklaring
i pointer: Tæller ydre runder (0 til n-2)
j pointer: Scanner elementerne (j vs j+1)
n pointer: Arrayets faste længde (index 6)
